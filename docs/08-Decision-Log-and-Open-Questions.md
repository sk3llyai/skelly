  - **Two DB roles.** The app runs as a restricted `skelly_app` role (Prisma `url` / `DATABASE_URL`) that is **subject to RLS** (not superuser, no `BYPASSRLS`); migrations/DDL run as the privileged `skelly` role (Prisma `directUrl` / `DATABASE_ADMIN_URL`). Set via the `withOrg(prisma, orgId, fn)` helper using `set_config(..., is_local => true)` so context is transaction-local and never leaks across pooled connections. Policies use `NULLIF(current_setting('app.current_org_id', true), '')::uuid` so an unset/empty context matches no rows (**deny by default**).

  - **Local Postgres on host port 5433** (not 5432) to avoid clashing with an existing local Postgres. `pgvector/pgvector:pg16` image (pgvector available for later).
